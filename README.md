# My Todo List - react-redux

react-redux로 Todo List 만들기

⚙ Features
* Create Todo
* Read Todos, Todo
* Update Todo
* Delete Todo

📌 Requirement
* redux를 이용한 전역 상태 관리
* todos 모듈은 Ducks 패턴으로 구현

* pages

  * Router.js
